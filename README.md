# tats
####By Danny Lujan

##TATS builds common apache rewrite rules

###Usage:
```
python tats.py 
```
Choose from menu:
```
       1.Force HTTPS (generic)
       2.Force www (generic)
       3.Redirect old domain to new
       4.Exit/Quit
```       
Option 2 and 3 will also allow for optional force https and option 3 will also allow for optional www forcing.
